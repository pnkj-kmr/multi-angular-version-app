# Angular 14 App

This is an Angular 14 application with user authentication and CRUD operations.

## Features

- Login and Signup pages
- User list with add, edit, and delete functionality
- Reusable components
- Authentication guard
- RESTful API integration

## Installation

```bash
npm install
```

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.
